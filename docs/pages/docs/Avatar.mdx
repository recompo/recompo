import Avatar from '../../../dist/Avatar';

### Description

Avatar component lets you manage your avatar in a powerful and unique ways

### Available Props
- Required
  - src &rarr; `string`
  - size &rarr; `string` 

- Optional
  - variant &rarr; `circular` `rounded` `square` default set to circular
  - alt &rarr; `string`

### Usage

```js
import { Avatar } from 'recompo';
// OR
import Avatar from 'recompo/Avatar';

return (
  <>
    <Avatar src="https://recompo.js.org/" alt="logo" size="100px" variant="rounded" >Hello World</Avatar>
  </>
);
```
